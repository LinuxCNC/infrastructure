EMC2: HEAD

The EMC2 tree is the focus of new development. The head of the tree is the mainstream version of EMC2. Other branches are used for experimental work, things that will break the main tree, or variations that need to remain independent of the main tree for one reason or another.

EMC2: TESTING

The EMC2 TESTING tree is stable development version. During the period prior to a release it is the release candidate (a tag on the release branch). After the release the TESTING tag moves back to the trunk and becomes the release candidate for the next release.

RCSLIB

The RCSLIB module is used by EMC1. No further development (except for minor bugfixes) is anticipated here. This module will remain in the compile farm for a while, but may eventualy be removed. Note that the realtime portions of RCSLIB do not (and will not) compile under kernel 2.6, so the results reported for BDI-4 systems are for the user space parts of RCSLIB only.
